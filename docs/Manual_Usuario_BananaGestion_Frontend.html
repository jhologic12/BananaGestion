1. Introducción

BananaGestion es una aplicación web diseñada para la gestión integral de fincas bananeras. Permite administrar trabajadores, lotes de terreno, labores culturales, registro de cosechas y control de inventario de insumos.

Requisitos:
  • Navegador moderno (Chrome, Firefox, Edge, Safari)
  • Conexión a internet para acceder al servidor
  • Cuenta de usuario activa proporcionada por el administrador

2. Menú Lateral y Navegación

Al iniciar sesión, el usuario ve una interfaz con un menú lateral izquierdo que contiene las siguientes opciones:

Descripción de la Pantalla

La pantalla de login es la página inicial de la aplicación. Presenta un diseño limpio y minimalista con el logotipo de BananaGestion.

Campos del Formulario

Campo Tipo Validación Descripción
Email Texto / Email Requerido, formato email Correo electrónico registrado en el sistema

Flujo de Uso

1
Ingrese su email
Escriba el correo electrónico asociado a su cuenta (ej: juan.perez@finca.com)
2
Ingrese su contraseña
Escriba su clave de acceso. Los caracteres se ocultan por seguridad.
3
Click en "Iniciar Sesión"
Se validarán las credenciales y se redireccionará al Dashboard si son correctas.
Mensajes de error comunes:
  • "Email requerido" - No se ingresó email
  • "Contraseña requerida" - No se ingresó contraseña
  • "Credenciales inválidas" - Email o contraseña incorrectos

Flujo para Crear Asignación

1
Haga clic en "Nueva Asignación"
Se abre un formulario modal.
2
Seleccione el tipo de labor
Ej: Fumigación, Riego, Abonado, Control de Malezas
3
Asigne un obrero
Seleccione el trabajador responsable de la tarea.
4
Seleccione el lote
Elija la parcela donde se realizará la labor.
5
Defina la fecha programada
Establezca cuándo debe realizarse la tarea.
6
Guarde la asignación
La tarea aparecerá en la lista de pendientes.

11. Permisos por Rol

El sistema BananaGestion implementa control de acceso basado en roles (RBAC). Cada usuario tiene permisos específicos según su rol:

Módulo Administrador Supervisor Obrero
Dashboard ✓ ✓ ✓
Usuarios - Ver ✓ ✓ ✗
Usuarios - Crear/Editar ✓ ✓ ✗
Usuarios - Eliminar ✓ ✗ ✗
Lotes - CRUD ✓ ✓ Ver
Labores - Ver ✓ ✓ ✓
Labores - Crear/Editar ✓ ✓ Solo Completar
Cosecha - Ver ✓ ✓ ✓
Cosecha - Registrar ✓ ✓ ✗
Inventario - Ver ✓ ✓ ✓
Inventario - Gestionar ✓ ✓ ✗
Reportes ✓ ✓ ✗
Configuración ✓ ✗ ✗
Recomendación: Asigne el rol mínimo necesario a cada usuario. Use "Administrador" solo para quienes gestionan el sistema, "Supervisor" para encargados de campo, y "Obrero" para trabajadores que solo ejecutan tareas.
